    public cloneableExtendedMixin: Object source 

    import {cloneableExtendedMixin} from '@ignavia/util/src/Cloneable.js'

    In addition to everything cloneableMixin offers this mixin also provides an easier to use interface. It adds methods with names that correspond to the keys in cloneableSymbols.

    Example:

    Object.assign(MyClass.prototype, cloneableExtendedMixin);
const myObject = new MyClass();
const clone    = myObject.clone();

    public cloneableMixin: Object source 

    import {cloneableMixin} from '@ignavia/util/src/Cloneable.js'

    A mixin for Cloneable behavior. The keys of the methods are the symbols defined in cloneableSymbols.

    Example:

    Object.assign(MyClass.prototype, cloneableMixin);
const myObject = new MyClass();
const clone    = myObject[cloneableSymbols.clone];

    public observableMixin: Object source 

    import {observableMixin} from '@ignavia/util/src/Observable.js'

    This is a mixin for Observable behavior if an EventManager is used. The event manager must be reachable by this.eventManager. This mixin can be used to facade the use of an EventManager.

    Example:

    Object.assign(MyClass.prototype, observableMixin);
let myObject = new MyClass(),
myObject[observableSymbols.addListener](() => console.log("Called"), "update");
