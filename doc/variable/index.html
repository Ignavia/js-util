    public observableMixin: Object source 

    import {observableMixin} from '@ignavia/util/src/Observable.js'

    This is a mixin for Observable behavior if an EventManager is used. Several assumptions are made: The event manager must be reachable by this.eventManager and the EventManager class must be imported under this exact name. This mixin can be used to facade the use of an EventManager.
