    TolkienMap

    Direct Subclass:

    Tolkien1To1Map, Tolkien1ToNMap, TolkienMToNMap

    An abstract class for bidirectional maps. The name is inspired by "The Hobbit or There and Back Again" by J. R. R. Tolkien.
