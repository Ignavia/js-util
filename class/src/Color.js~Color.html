    public toHSLA(): Object source

    Converts this Color object to HSLA.

    Return:

    Object

    An HSLA representation of this color. The resulting object has the three properties h, s and l for the hue, saturation and lightness respectively. The hue is a number in the range [0, 360) and saturation and lightness are in the range [0, 1]. It also has a property for the alpha value. This is a number in the range [0, 1]. 0 represents full transparency and 1 is full opacity.

    public toRGBA(): Object source

    Converts this Color object to RGBA.

    Return:

    Object

    An RGBA representation of this color. The resulting object has the three properties r, g and b for the red, green and blue channel respectively. Those are integers in the range [0, 255]. It also has a property for the alpha value. This is a number in the range [0, 1]. 0 represents full transparency and 1 is full opacity.
